# CD Catalog Web Application

A simple portfolio project showcasing a **CD catalog website** built with **Laminas Skeleton Application**.  

## Features

- Fully functional **backend** with database support.
- Manage CD information (add, edit, delete) through the backend.
- Interact with CD data on the website frontend.
- Demonstrates proficiency with Laminas framework and MVC architecture.

## Tech Stack

- PHP 8+
- Laminas Framework (Skeleton Application)
- SQLite3 for database
- HTML/PHP for frontend

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/cd-catalog.git
2. Install dependencies using Composer:
   ```bash
   composer install
